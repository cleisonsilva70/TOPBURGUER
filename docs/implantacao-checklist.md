# Checklist de Implantacao por Cliente

## Dados da loja

- nome oficial
- nome curto
- telefone
- WhatsApp
- endereco
- horario

## Marca

- logo
- cores principais
- slogan ou frase principal

## Cardapio

- categorias cadastradas
- ordem das categorias
- produtos com preco
- descricoes revisadas
- imagens revisadas
- destaques marcados
- tamanhos configurados
- opcionais configurados

## Promocoes

- banner principal
- banners secundarios
- ordem dos banners
- selo de campanha
- datas de inicio e fim

## Operacao

- senha da cozinha
- fluxo de atendimento validado
- som da cozinha testado
- modo tablet revisado

## Publicacao

- deploy publicado
- links testados
- acesso interno validado
- pedido real de teste concluido
