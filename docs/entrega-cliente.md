# Modelo de Entrega para o Cliente

Segue o acesso ao cardapio digital da sua hamburgueria.

- Login interno: `COLE_AQUI`
- Atendimento: `COLE_AQUI`
- Cozinha: `COLE_AQUI`

## Senha interna

- `DEFINIR_SENHA`

## Como funciona

1. O cliente faz o pedido pelo cardapio.
2. O pedido chega no WhatsApp da loja.
3. O atendimento confirma o pagamento.
4. A cozinha recebe somente os pedidos liberados.

## Recomendacoes

- teste um pedido completo antes de iniciar o atendimento real
- mantenha banners e produtos atualizados
- troque a senha quando houver mudanca de equipe
